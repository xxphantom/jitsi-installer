# Jitsi Meet Docker Installer

[![README in English](https://img.shields.io/badge/README-EN-blue)](README.md)

Интерактивный bash-скрипт для установки, настройки и управления Jitsi Meet на Docker.

## Быстрый старт

```bash
sudo bash -c "$(curl -sL https://raw.githubusercontent.com/xxphantom/jitsi-installer/main/installer.sh)"
```

## Возможности

- Установка Jitsi Meet одной командой через Docker
- Интерактивный мастер настройки
- Автоматическая генерация SSL-сертификата (Let's Encrypt)
- Встроенное управление пользователями (создание, удаление, список, смена пароля)
- Двуязычный интерфейс (русский / английский)
- Управление контейнерами (перезапуск, просмотр логов)
- Полное удаление

## Требования

- Linux на базе Ubuntu/Debian
- Права root (sudo)
- Доменное имя, направленное на сервер
- Открытые порты 80, 443, 10000/udp

## Параметры установки

| Параметр | Описание |
|----------|----------|
| `--lang=ru` | Русский интерфейс (по умолчанию) |
| `--lang=en` | Английский интерфейс |

## Ручная установка

Если предпочитаете клонировать репозиторий:

```bash
git clone https://github.com/xxphantom/jitsi-installer.git
cd jitsi-installer
sudo ./install_jitsi.sh
```

## Пункты меню

1. **Установить Jitsi Meet** — Полная установка с настройкой домена, часового пояса и аутентификации
2. **Удалить Jitsi Meet** — Удаление всех компонентов и конфигурации
3. **Перезапустить Jitsi Meet** — Перезапуск Docker-контейнеров
4. **Просмотр логов** — Отображение логов контейнеров
5. **Управление пользователями** — Создание, удаление, список пользователей, смена паролей
6. **Показать учётные данные** — Отображение содержимого /opt/jitsi/credentials.txt

## Расположение файлов

| Путь | Описание |
|------|----------|
| `/opt/jitsi` | Основная директория установки |
| `/opt/jitsi/.jitsi-meet-cfg` | Файлы конфигурации |
| `/opt/jitsi/credentials.txt` | Учётные данные администратора |
